Discrete Fourier Transform
Direct Computation

C program that computes the discrete Fourier transform directly. The program takes a list of values in the sampled domain and outputs a vector of Fourier coefficients in the frequency domain. The direct computation uses a DFT matrix to transform the real samples into a complex vector. This algorithm runs in O(n^2) where n is the length of the input vecotr. For large n, a fast Fourier transform algorithm (Cooley-Tukey) is prefered as it runs in O(n*log(n)).
